Why Kinetrix?

Designed for the edge. Optimized for control.

Hardware Native

01. Installation

git clone https://github.com/M4dM0nk3y/Kinetrix.git
cd Kinetrix
./install.sh

02. Logic & Control

main.knx
# Read Sensor and Actuate
make var dist = read analog pin 0

if dist < 20 {
    turn on 13
    servo 9 set 180
} else {
    turn off 13
    wait 100
}

03. Inverse Kinematics

arm.knx
# Calculate Joint Angles
make var target_x = 50.5
make var theta = acos(target_x / 100)
make var servo_val = theta * 180 / 3.14159
